# Engineering Internship Tracker

Automated tool that aggregates engineering and robotics internship postings, filters relevant roles, and displays results in an interactive dashboard.

## Highlights

- Aggregates listings from Greenhouse & Lever job boards  
- Filters Mechanical, Robotics, Mechatronics, and Software roles  
- Removes duplicate postings and tracks new opportunities  
- Interactive Streamlit dashboard for browsing and filtering  
- Exportable dataset of internship opportunities  

## Tech Stack

Python • Requests • Pandas • Streamlit • REST APIs • CSV/JSON
